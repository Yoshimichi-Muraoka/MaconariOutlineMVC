【アプリケーション名】
マコなり社長アウトライン作成システム

【概要】
https://www.youtube.com/watch?v=ArMNB_QTSXw
上記のURLの動画内で言及しているアウトラインを作成するための、フォーマットとして使用できるアプリケーションです。
簡単に言えばタスク管理のようなものですが、普通のタスク管理より便利で、私もプライベートで使用しております。

アプリにログインすると、ページ毎に一言説明が書いてあるので、動画を観ていなくても利用出来ます。記述例も載せてあります。
ログインにはIDとパスワードが必要になりますが、初期値が入力してあるので、そちらを利用すればどういった機能が実装されているか、
一通り御覧になれます。
自分だけのアウトラインを作成したい場合は新規会員登録を行い、改めてログインしてください。
新規会員登録に必要なのは半角英数字4文字のIDとパスワードです。個人情報などの入力は必要ありません。

【実装機能】
ログイン認証/新規会員登録/退会/アウトライン一覧表示/新規アウトライン登録/アウトライン編集/アウトライン削除
カレンダー/スケジュール登録/スケジュール編集/スケジュール削除/登録したスケジュールをスケジュール欄に反映/登録したスケジュールをカレンダーに反映
